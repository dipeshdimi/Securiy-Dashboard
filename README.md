### Installation

1. **Clone the repository:**

    ```bash
    git clone https://github.com/dipeshdimi/Securiy-Dashboard.git
    ```

2. **Install dependencies:**

    ```bash
    npm install
    ```

3. **Start the app:**

    ```bash
    npm run dev
    ```
